# Admin FE (React + Vite)

# Setup
 1. Mohon gunakan setidaknya gunakan node v16.20.2
 2. Pastikan vite telah terinstall (npm install -g create-vite)
 3. cd ke project
 4. npm install
 5. npm run dev
 6. hit http://127.0.0.1:5173
